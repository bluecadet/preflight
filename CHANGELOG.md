# Changelog

## [2.0.0-beta.5](https://github.com/bluecadet/preflight/compare/preflight-v1.0.0-beta.5...preflight-v2.0.0-beta.5) (2026-07-09)


### ⚠ BREAKING CHANGES

* **inventory:** an omitted transport now resolves to ssh on port 22 with host_key_policy accept-new pinning the remote key on first connect, instead of winrm on port 5985. Inventories that relied on the implicit WinRM default must set transport: winrm explicitly.
* **stdlib:** preflight/windows-power no longer accepts scheduled_reboot_* inputs or creates a scheduled reboot task. Use the scheduled_task module instead.
* **target:** remove SYSTEM become support
* inventory.yml and --inventory are no longer supported. Project inventory now lives under the inventory field in preflight.yml with hosts as the primary list.
